# 实现功能： #
    1. 可依次选择进入各子菜单进行查询
    2. 可从任意一层往回退到上一层
    3. 可从任意一层退出程序
# 开发环境： #
	Python 3.5.2
# 启动方式： #
    打开CMD
	进入到3menu.py文件所在的目录
	执行Python 3menu.py
# 登录信息： #
    不需要登陆
