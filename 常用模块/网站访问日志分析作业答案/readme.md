# 实现功能： #
    程序启动后，给用户提供接口，实现网站访问日志分析
    1 统计本日志文件的总pv、uv
    2 列出全天每小时的pv、uv数
    3 列出top 10 uv的IP地址，以及每个ip的pv点击数
    4 列出top 10 访问量最多的页面及每个页面的访问量
    5 列出访问来源的设备列表及每个设备的访问量
# 开发环境： #
	Python 3.5.2
# 启动方式： #
    打开CMD
	进入到Website analysis.py文件所在的目录
	执行Website analysis.py
	配套文本文件为网站访问日志
# 登录信息： #
    不需要登陆
